This page demonstrates how to embed filtered views of the Psychedelic Ecosystem Map using URL parameters. The embed mode shows only the visualization without navigation or page chrome, perfect for embedding on other sites.

üìù Available URL Parameters:

  • embed=true - Enables embed mode (hides navigation and header)
  • location=Colorado - Filters to specific location
  • section=organizations or section=projects - Sets active tab
  • view=grouped or view=geographic or view=table - Sets view mode

Example 1: Colorado Organizations (Grouped by Ecosystem Role)

Shows only Colorado-based organizations in the bubble visualization:

http://localhost:3000/ecosystem-map?embed=true&location=Colorado§ion=organizations&view=grouped

Example 2: Colorado Geographic Composition

Geographic visualization of Colorado organizations:

http://localhost:3000/ecosystem-map?embed=true&location=Colorado§ion=organizations&view=geographic

Example 3: All Organizations (No Location Filter)

Full ecosystem map without location filtering:

http://localhost:3000/ecosystem-map?embed=true§ion=organizations&view=grouped

Example 4: Projects View

Projects instead of organizations:

http://localhost:3000/ecosystem-map?embed=true§ion=projects&view=grouped

üí° How to Use in Production

Replace localhost:3000 with psychedelic-ecosystem-map.vercel.app in the URLs above:

https://psychedelic-ecosystem-map.vercel.app/ecosystem-map?embed=true&location=Colorado§ion=organizations&view=grouped
